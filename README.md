# TodoList 案例
这是自己在学习react的过程中做的一个非常典型的小案例，不过加了一些自己的特色，比如列表动画等...

## 主要技术栈
- React
- TypeScript
- Vite
- React-Transition-Group

## 安装
拉取代码
```bash
git clone https://github.com/Chihiro1221/react-todo-list.git
```
安装依赖
```bash
pnpm install
```
运行项目
```bash
# 默认运行到3000端口
pnpm run dev
```
